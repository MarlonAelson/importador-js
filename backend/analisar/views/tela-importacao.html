Atencão!!! Antes de importar os dados do EXCEL trate/verifique:
  1. Verifique se o arquivo (planilha) está com a extensão XLSX. Observação: Caso o arquivo esteja em XLS, apenas renomear para XLSX não dará certo;
  2. Verifique se a planilha está nomeada igual ao nome da tabela do banco de dados e com extensão XLSX. Exemplo: linhas.xlsx, produtos.xlsx;
  3. Verifique se as colunas das planilhas estão nomeadas iguais as colunas da tabela do banco de dados. Exemplo: codigo_linha, codigo_aliquota etc;
  4. Deixe apenas as colunas que de fato tem informação, as demais remova. Exemplo: A planilha tem a coluna referência, mas nenhuma célula está preenchida;
  5. Trate os dados deixando no tamanho máximo comportado pela coluna que irá receber a informação no banco de dados. Exemplo: Descrição de Produtos só comporta 50 caracteres;
  6. Trate os dados deixando no tamanho máximo comportado pela coluna que irá receber a informação no banco de dados. Exemplo: Descrição de Produtos só comporta 50 caracteres;
